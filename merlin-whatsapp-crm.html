🔌 与 Merlin 系统对接说明

1数据共享机制

此 CRM 页面与 merlin-admin.html 和 index.html 同域部署时，通过 localStorage 和 Firebase Firestore 自动共享顾客数据，无需额外配置。

2在 index.html 注册回调（推荐）

在 index.html 的 notifyRexOnFlowComplete 函数末尾添加以下代码，当顾客完成问卷时立即触发自动回复检查：

// 在 notifyRexOnFlowComplete() 函数末尾加入： window.dispatchEvent(new CustomEvent('merlin:lead_saved', { detail: { lead: leadData, flow: _flow } }));

3Firebase 实时监听（已内置）

此页面已连接 Firebase Firestore（merlin-crm-2），会实时监听 merlin_leads 集合的新增记录并自动触发回复逻辑。

4三个文件放同一目录

your-site/ ├── index.html ← Merlin 聊天机器人（前台） ├── merlin-admin.html ← CRM 管理后台 └── merlin-whatsapp-crm.html ← WA 自动回复（本页面）
